# MMDDanceHoloLens

Unity | HoloLensでMMDモデルを利用する

## 環境

Unity 2019.4.18f1 <br>
HoloLens 2 <br>
MixedRealityToolkit 2.6.1 <br>
[MMD4Mecanim Beta 20200105](http://stereoarts.jp/) <br>

## 実行

UnityのPlayボタンから再生
