# MMDDanceHoloLens

Unity | HoloLensでMMDモデルを利用する

## 環境

Unity 2019.4.18f1
HoloLens 2
MixedRealityToolkit 2.6.1
[MMD4Mecanim Beta 20200105](http://stereoarts.jp/)

## 実行

UnityのPlayボタンから再生
